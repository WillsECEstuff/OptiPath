std::tuple< float, float > Database::getProductPosition ( std::string  productID)

This function returns a tuple<float, float> in the xy coordinate format given a particular productID. Before using this function, the database should be populated by using the populateDatabase function.

Parameters

Returns
This function returns a tuple<float, float> in the xy coordinate. If a product does not exist in the database, the function will return a tuple of <-1, -1>.
